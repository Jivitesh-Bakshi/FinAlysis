Financial Analysis tool built to let one visualise daily expenditure and keep track, supported by openAI chatbot! 

The front end is complete and ready to read log files with transaction details in the specified format, a demo file has been provided to test the same. The Ai chatbot can be used by anyone with an OpenRouter key, which can be generated from the OpenRouter website. 

The next steps in the process would be creating a Gpay notification scraper for android, allowing it to read outgoing transactions and automatically update log files which shall be internally passed on to the existing front end but made compatible with android and the whole package being converted and compiled as a ".apk" file.
